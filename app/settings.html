Call transcripts (Fireflies)

Calls are pulled from Fireflies automatically every 45 minutes. For instant processing, Reinis can additionally save this webhook.
Send the URL to Reinis: app.fireflies.ai → Settings → Developer settings → Webhook, paste, save. Done once.

Team alerts (Slack)

Booked-lead pings + the morning call sheet. Paste a Slack incoming-webhook URL.
To create one:
  1. api.slack.com/apps → Create App → From scratch → pick your workspace
  2. Incoming Webhooks → activate → Add New Webhook → choose the channel
  3. Copy the URL here, Save, then Test
